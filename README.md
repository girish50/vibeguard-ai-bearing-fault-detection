
# 🔧 VibeGuard AI

AI-based system for bearing fault detection and predictive maintenance.

---

## 🚀 Overview

VibeGuard AI analyzes vibration signals to detect faults and estimate machine health.
It uses a hybrid machine learning approach to identify early signs of failure and support preventive maintenance.

---

## ✨ Features

- 🟢 Fault detection (Normal / Warning / Fault)
- ⚠️ Early warning before failure
- 🧠 Health score (0–100)
- 📉 Anomaly detection from signal data
- 🌐 Interactive web app (Streamlit)

---

## ⚙️ How It Works

Signal → Features → Anomaly Detection → Classification → Result

---

## 🧠 Tech Stack

- Python
- Scikit-learn
- NumPy, Pandas
- Streamlit

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
